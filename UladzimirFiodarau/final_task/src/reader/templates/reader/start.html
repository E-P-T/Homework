Welcome to RSS Reader web service


The RSS Reader has been created as a final task for graduating EPAM March 2022 Python Foundation course. Following functions are currently implemented:

- One-shot conversion from RSS to Human readable format
- Server-side news caching in database and filesystem caching for faster response
- Adding new RSS sources to tracking (cache) using API
- Updating tracked (cached) RSS sources using API
- Conversion of news to HTML and PDF formats
- User registration and authentication using email
- Sending a greeting email to user after registration
- User settings changes and user deletion with success messages
- Customized error handlers and messages.error usage
- Navigation bar and favicon
